Lone Warrior

Lone Warrior is an introduction to creating open-world adventure games in Unreal Engine 5 and C++. This is currently a work in progress, however, I have learned many valuable skills in terms of game development. I've developed an understanding for creating character models, animations, input actions, character rigs, maps, world partitioning (for performance), implementing character functionality, and many more useful skills. Currently, I am working on implementing free-roaming wildlife that the user can hunt or capture (mounts), mobs that attack the user if they are within a certain range, multi-level functionality, and a upgradeable skills system (with skill points).

Quick Gameplay

As you can see, this is certainly still a work in progress. The user functionality has been implemented, however, it is still a barebones game without much objective.

All source code is available upon request.
